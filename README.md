How to get Libraly

Step 1. Add the JitPack repository to your build file

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
 
 Step 2. Add the dependency
 
 dependencies {
	        implementation 'com.github.TonRobotic:CircleProgressBar:Tag'
	}
